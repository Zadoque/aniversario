# 🎂 aniversario

Página de convite digital para o aniversário de **Zadoque Carneiro**,
hospedada em [zadoque.github.io/aniversario](https://zadoque.github.io/aniversario).

## ✨ Sobre

Convite interativo feito em HTML + CSS + JS puro, sem dependências de build.
Desenvolvido para ser compartilhado via WhatsApp e aberto diretamente no celular.

## 🎯 Funcionalidades

- **Countdown** em tempo real até o início da festa
- **Integral matemática** no lugar da idade — com botão para revelar a resolução passo a passo
- **Fotos da família** embutidas diretamente no arquivo (base64)
- **Cardápio digital** via link do Anota Aí
- **Mapa com pin exato** do local (coordenadas GPS precisas)
- **Botão de confirmação** via WhatsApp com confetti 🎉
- 100% responsivo e mobile-first

## 🛠️ Tecnologias

- HTML5 + CSS3 + JavaScript vanilla
- [KaTeX](https://katex.org/) — renderização da integral matemática
- [Google Fonts](https://fonts.google.com/) — Playfair Display + Inter
- Google Maps Embed API
- GitHub Pages — hospedagem gratuita

## 📁 Estrutura

```
aniversario/
└── index.html   ← arquivo único, tudo embutido
```

## 🚀 Deploy

```bash
git clone https://github.com/zadoque/aniversario.git
cd aniversario
# substitua o index.html e faça push
git add index.html
git commit -m "atualiza convite"
git push
```

O GitHub Pages publica automaticamente a partir da branch `main`.
Acesse em: **zadoque.github.io/aniversario**
